# webpack-starter

Проект можно использовать, как базу для создания новых приложений. Включает настроенный линтер, сборщик и dev-сервер.

## Установить зависимости
```
npm i
```

### Запустить dev-сервер
```
npm run serve
```

### Собрать в прод
```
npm run build
```

### Проанализировать код
```
npm run lint
```

## Dev-зависимости

Вебпак
* webpack
* webpack-cli
* webpack-dev-server
* webpack-merge

Плагины
* clean-webpack-plugin
* html-webpack-plugin

Линтер
* eslint

Стили
* node-sass
* sass-loader
* style-loader
* css-loader
